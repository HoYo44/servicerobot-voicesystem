## Setup Guide
This setup guide assumes Pyhon3.8.
- DeepSpeech does not support versions later than Python3.8 (2023/08/23)
### Rasa
install Rasa
```
pip install rasa
```

### DeepSpeech
install DeepSpeech
```
pip install deepspeech
```
- If the installation fails, upgrading the pip version will solve the problem.
